# Git과 Git hub
---
**Git**: 컴퓨터 파일의 변경 사항을 추적하고 여러 명의 사용자들 간에 해당 파일들의 작업을 조율하기 위한 스냅샷 스트림 기반의 분산 버전 관리 시스템   

**Git hub**: 분산 버전 관리 툴인 깃 저장소 호스팅을 지원하는 웹 서비스

(출처: 위키백과)

한 마디로 정리하면 Git을 통해 순간 순간의 버전을 Git hub로 올리고 Git hub에서 저장된 파일들을 확인, 공유할 수 있다.
